Derlenen ve Yorumlanan programlama dillerinin geliştirme farklarını irdeleyiniz.

Yorumlanan dillerde yorumlayıcı çalışırken yazılan kodu satır satır yorumlarken; derlenen dillerde çıktı olarak bir uygulama üretlilir (Hesap makinesi, oyun vb.)
Bu sebeple yorumlanan dillerle yazulan uygulamalar, derlenen dillerle üretilen uygulamalara göre genelde daha fazla bellek alanı kaplar ve görece daha yavaş çalışır.

While döngüsü örneği:

```c
#include <stdio.h>

/* print Fahrenheit-Celsius table for fahr = 0, 20, ..., 300 */
int main(void)
{
        int fahr, celsius;
        int lower, upper, step;

        lower = 0;   /* lower limit of temperature scale */
        upper = 300; /* upper limit */
        step = 20;   /* step size */

        fahr = lower;
        while (fahr <= upper) {
                celsius = 5 * (fahr-32) / 9;
                printf("%d\t%d\n", fahr, celsius);
                fahr = fahr + step;
        }

        return 0;
}
```

Sizde sadece toplama operatörü `+` ve `while` döngüsü kullanarak 2 sayıyı çarpınız.

Çözümü

```c
#include <stdio.h>

int main() {
    int a, b;
    int sonuc = 0;
    int i = 0;

    printf("Iki sayi girin: ");
    scanf("%d %d", &a, &b);

    while (i < b) {
        sonuc = sonuc + a;
        i = i + 1;
    }

    printf("Sonuc: %d\n", sonuc);

    return 0;
}
```
